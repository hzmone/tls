# TLS | Transaction Ledger System
### The easy to learn version control system, made fully in rust.
**How to use:**
-
### Initialize your project's root directory:
> ```
>tls -i
>```
>or
>```
>tls --init
>```
### Stage your files:
>```
>tls -a ./
>```
>or 
>```
>tls --add ./
>```
### Commit your changes:
>```
>tls -c "a commit message"
>```
>or 
>```
>tls --commit "a commit message"
>```
